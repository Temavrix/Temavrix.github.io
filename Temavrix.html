We Know Design

Our startup aims to make our apps more vibrant, sustainable and efficient.
These days whenever a product is being created, developers need to not only think about the backend code but also now need to think about integrating the backend code to the frontend UI/UX buttons and funtions so that the users can navigate through the product using the frontend without prior knowledge or understanding of the Backend coding.

What Our organisation has done so far

Our Organisation uses Visual Studios Code to code and Git for version control for most of our projects and endeavours. While we want to create successful products we would need to start somewhere to show the world that we are capable of creating, testing and deploying products and also show that we have the ability to also market our products to our indented targeted audience. We also place our product's code in Github for our coding audience to see and also help contribute ideas and test it to find bugs for us to build a better version of our product.

JARVIS
Started in 2020, this Chatbot built using python and pytorch is capable of chatting and retrieving information, daily news for it's user. It's bundled with a sleek FrontEnd using Tkinter for better User Experience (UX) and implements a Feed Forward Neural Network with 2 hidden layers where users can customize for their own use case by updating the Json file. This project is our First Flagship Project and has gotten 70+ stars on github so far.

NexaView
Announced in 2022 as Temavrix's most ambitious project, NexaView is an All-in-one Web-based Dashboard for Weather and Weather Forecasts for their current location , News headlines for different topics from different countries and add and track tasks to keep themselves productive stored in firebase. This project utilizes React Native and TailwindCSS for the frontend and Firebase as the backend database while using Redis for caching frequently used data.

PatchFlow
Announced in end-2025, PatchFlow is a desktop-based issue tracking and developer productivity tool built using JavaFX and SQLite, designed to help developers manage bugs, track issues, and improve workflow efficiency. It's Simplicity, Speed, Clean UI and Developer-first features make this suitable and lightweight for solo-developers.
